# Proyecto 1
## Andrés Ramírez Ruiz 

## Objetivo
Construir una calculadora funcional aplicando:
- Manipulación del DOM
- Manejo de datos primitivos en JS
- Manipulación de strings
- Uso básico de regex
- Estructuración semántica con HTML
- Estilado con CSS

## Entregable
Repositorio en GitHub que incluya:
- index.html
- styles.css
- script.js
- Operaciones básicas

## Tu calculadora debe soportar:
- Suma (+)
- Resta (-)
- Multiplicación (*)
- División (/)
- Exponencial
- Operaciones con valores negativos
- Operaciones con punto decimal
- Seccion de Operaciones anteriores
 

## Criterio	                            Peso
- Funcionalidad	                        40%
- Lógica JS (sin eval)	                25%
- Validaciones (regex + lógica)	        15%
- UI/UX	                                10%
- Organización del código	            10%
